Where Seattle’s upzones would take place

TThe Seattle City Council is considering legislation that would make changes in 27 urban villages and some other areas while imposing new requirements on developers to help create affordable housing. The changes would occur across various density levels. Most land now zoned exclusively for single-family homes would not be affected.
